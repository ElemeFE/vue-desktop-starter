# vue-desktop-starter
A starter for vue-desktop.

Used plugins:
- vue-i18n
- vue-resource
- vue-router

# Usage

```Bash
# Install dependencies
npm install

# Run webpack dev server
npm run dev

# Run webpack with watch option
npm run watch

# Run webpack in production mode(use UglifyJs)
npm run build

# Run webpack in deploy mode(use UglifyJs and set public-path)
npm run deploy
```
